# Trivandrum Hidden Food Spots

A modern web application showcasing hidden food destinations across Trivandrum.

## Features

- Search hidden food spots
- Browse by category
- Responsive design
- Mobile-friendly interface
- Interactive user experience

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Folder Structure

```
.
├── index.html
├── favicon.png
├── bg.png
├── assets/
│   ├── index-local.css
│   ├── index-local.js
│   └── enhancements.js
```

## Running Locally

Open `index.html` in any modern web browser.

## GitHub Pages

The website is ready to be deployed through GitHub Pages.

Repository Settings → Pages → Deploy from Branch → main → /(root)
